


Serket

- ``serket`` aims to be the most intuitive and easy-to-use neural network library in ``jax``.
- ``serket`` is fully transparent to ``jax`` transformation (e.g. ``vmap``, ``grad``, ``jit``,...).


Installation
------------

Install from pip::

   pip install serket
